# Release Notes

## v1.0.0 - 2026-03-19

TranscribeBench now has a stable `v1.0.0` baseline for the current local benchmarking workflow.

### Highlights
- Added a dated archived benchmark snapshot for the 200-sample Dutch baseline.
- Added a dated benchmark config for reproducible archived benchmark runs.
- Clarified the difference between mutable `latest` outputs and archived benchmark artifacts.
- Established `v1.0.0` as the current release baseline for future roadmap and release planning.
